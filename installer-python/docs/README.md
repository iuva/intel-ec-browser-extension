# Chrome 浏览器扩展安装器 - 源代码目录结构说明

## 目录概述

`installer-python` 目录包含了浏览器扩展的 Windows 安装器程序，采用 Python + Tkinter 开发，提供图形化的安装向导界面。

## 目录结构说明

### 📁 icon/
**图标资源目录**
- 包含安装器程序的多尺寸图标文件
- 支持不同分辨率的显示需求

### 📁 output/
**输出目录**
- 存储打包生成的安装器可执行文件
- 包含最终发布的安装程序

### 📁 setp/
**安装步骤图片目录**
- 存储安装向导界面的步骤示意图
- 包含安装过程的视觉引导图片

### 📄 根目录文件
- **main.py** - 安装器主程序，包含图形界面和安装逻辑
- **build.py** - 打包脚本，用于将 Python 程序打包为 exe 文件
- **installer.spec** - PyInstaller 打包配置文件
- **requirements.txt** - Python 依赖包列表
- **README.md** - 项目说明文档

## 安装器功能特性

### 🎯 核心功能
- ✅ 图形化安装向导界面
- ✅ 自动检测 Chrome 浏览器
- ✅ 扩展文件复制和配置
- ✅ 注册表配置和快捷方式创建
- ✅ 安装进度显示和状态反馈

### 🔧 技术特点
- 基于 Python + Tkinter 开发，轻量级且跨平台兼容
- 支持 Windows 系统环境检测
- 提供完整的错误处理和用户反馈
- 支持静默安装和自定义安装路径

## 开发环境要求

### Python 环境
- Python 3.7+
- 依赖包：见 requirements.txt

### 开发工具
- PyInstaller（用于打包）
- PIL/Pillow（图像处理）
- Tkinter（GUI 界面）

## 打包和发布

### 打包流程
1. 运行 `python build.py` 进行打包
2. 生成的 exe 文件位于 `output/` 目录
3. 测试安装器功能完整性
4. 发布到目标平台

### 发布说明
- 安装器支持 Windows 7/10/11 系统
- 需要管理员权限进行安装
- 支持 Chrome、Edge 等基于 Chromium 的浏览器

## 安装流程

1. **环境检测** - 检查系统环境和浏览器
2. **路径选择** - 选择安装目录
3. **文件复制** - 复制扩展文件
4. **配置注册** - 配置浏览器扩展
5. **完成安装** - 显示安装结果

## 错误处理

安装器包含完善的错误处理机制：
- 权限不足时的友好提示
- 文件复制失败的重试机制
- 浏览器检测失败的处理方案
- 安装过程中的状态监控
