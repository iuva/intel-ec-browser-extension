# 安装和使用指南

## 快速开始

### 系统要求
- **操作系统**：Windows 7/10/11（64位）
- **浏览器**：Chrome 或基于 Chromium 的浏览器（Edge、Brave等）
- **权限**：需要管理员权限进行安装
- **磁盘空间**：至少 50MB 可用空间

### 安装步骤

#### 方法一：使用预编译安装器（推荐）

1. **下载安装器**
   - 从发布页面下载 `ec-chrome-extension-installer.exe`
   - 文件位于 `installer-python/output/` 目录

2. **运行安装器**
   - 双击运行安装程序
   - 如有安全提示，选择"运行"或"更多信息→仍要运行"

3. **跟随安装向导**
   - **步骤1**：阅读并接受许可协议
   - **步骤2**：选择安装目录（默认：`C:\ec-chrome-extension`）
   - **步骤3**：检测并配置浏览器
   - **步骤4**：等待文件复制完成
   - **步骤5**：确认安装完成

4. **完成安装**
   - 重启浏览器使扩展生效
   - 在浏览器工具栏中查看扩展图标

#### 方法二：从源代码安装（开发者）

1. **准备环境**
   ```bash
   # 安装 Python 3.7+
   # 安装依赖包
   pip install -r requirements.txt
   ```

2. **运行安装器**
   ```bash
   # 直接运行 Python 脚本
   python main.py
   ```

3. **打包发布**
   ```bash
   # 打包为可执行文件
   python build.py
   # 生成的文件在 output/ 目录
   ```

## 使用说明

### 扩展功能
安装完成后，扩展提供以下功能：

#### 1. VNC 远程连接
- 管理多个 VNC 主机连接
- 支持快速连接和断开
- 显示连接状态和性能信息

#### 2. 主机列表管理
- 添加、编辑、删除 VNC 主机
- 支持主机分组和搜索
- 导入/导出主机配置

#### 3. 设置和配置
- 自定义连接参数
- 配置通知和提醒
- 管理用户偏好设置

### 界面操作

#### 弹出窗口
- 点击浏览器工具栏中的扩展图标
- 查看快速连接和状态信息
- 进行常用操作和设置调整

#### 主界面
- 通过弹出窗口中的"打开主界面"进入
- 查看完整的主机列表和详细信息
- 进行复杂的管理操作

## 故障排除

### 常见问题

#### 1. 安装失败
**问题**：安装过程中出现错误提示
**解决**：
- 确保有管理员权限
- 关闭杀毒软件临时重试
- 检查磁盘空间是否充足
- 查看安装日志获取详细信息

#### 2. 扩展未显示
**问题**：安装完成后浏览器中看不到扩展
**解决**：
- 重启浏览器
- 在浏览器设置中启用扩展
- 检查扩展是否被浏览器阻止

#### 3. 连接失败
**问题**：VNC 连接无法建立
**解决**：
- 检查网络连接是否正常
- 确认 VNC 服务器运行状态
- 验证用户名和密码是否正确
- 检查防火墙设置

#### 4. 性能问题
**问题**：扩展运行缓慢或卡顿
**解决**：
- 关闭其他不必要的扩展
- 清理浏览器缓存
- 更新浏览器到最新版本

### 日志和调试

#### 查看安装日志
安装器会在以下位置生成日志文件：
```
C:\ec-chrome-extension\logs\install.log
```

#### 启用调试模式
开发者可以启用调试模式获取详细信息：
```python
# 在 main.py 中设置调试标志
DEBUG = True
```

## 卸载说明

### 方法一：使用安装器卸载
1. 重新运行安装器程序
2. 选择"卸载"选项
3. 确认卸载操作
4. 重启浏览器完成卸载

### 方法二：手动卸载
1. **删除扩展文件**
   ```
   C:\ec-chrome-extension\
   ```

2. **清理浏览器配置**
   - 在浏览器中禁用或删除扩展
   - 清理浏览器缓存和数据

3. **删除注册表项**（可选）
   ```
   HKEY_CURRENT_USER\Software\ec-chrome-extension
   ```

## 更新说明

### 自动检测更新
扩展会定期检查新版本，如有更新会提示用户。

### 手动更新
1. 下载最新版本的安装器
2. 运行安装程序
3. 选择"更新"选项
4. 跟随更新向导完成操作

### 版本兼容性
- 小版本更新通常保持向后兼容
- 大版本更新可能需要重新配置
- 更新前建议备份重要配置

## 安全说明

### 数据安全
- 连接密码等敏感信息会加密存储
- 不会收集用户的个人隐私数据
- 所有网络通信使用安全协议

### 权限说明
扩展需要以下权限：
- **存储权限**：保存配置和主机信息
- **网络权限**：建立 VNC 连接
- **标签页权限**：在浏览器中显示界面

### 安全建议
- 定期更新扩展版本
- 使用强密码保护 VNC 连接
- 在可信网络环境中使用
- 及时报告安全漏洞
