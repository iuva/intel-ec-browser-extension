# 页面模块说明

## 目录功能概述

`pages/` 目录包含了浏览器扩展的各个页面组件，每个页面对应扩展的一个独立功能界面。

## 文件说明

### 🪟 Popup.vue
**弹出窗口页面**

**功能描述：**
- 浏览器工具栏点击后显示的弹出窗口
- 提供快捷操作和基本信息展示
- 支持快速连接和设置调整

**页面特性：**
- 紧凑的界面设计，适合小窗口显示
- 实时显示扩展状态和版本信息
- 提供常用功能的快捷入口
- 支持主题切换和个性化设置

**主要功能区域：**
1. **头部区域** - 扩展名称、版本号、设置按钮
2. **快速连接** - 最近连接的主机列表
3. **状态显示** - 当前连接状态和系统信息
4. **操作按钮** - 打开主界面、刷新、帮助等

**交互功能：**
- 点击主机项快速建立 VNC 连接
- 支持搜索和筛选主机列表
- 提供设置界面的快捷入口
- 显示实时通知和状态更新

**使用场景：**
- 快速查看扩展状态
- 快速连接常用主机
- 进行简单的设置调整
- 查看通知和系统信息

## 页面设计原则

1. **简洁明了**：界面设计简洁，信息展示清晰
2. **快速响应**：页面加载和交互响应迅速
3. **功能聚焦**：每个页面专注于特定功能
4. **用户体验**：考虑用户操作习惯和便捷性

## 响应式设计

- 适配不同尺寸的弹出窗口
- 支持鼠标和触摸操作
- 考虑高DPI屏幕的显示效果
- 支持键盘快捷键操作

## 扩展建议

1. **新页面**：根据功能需求添加新的页面组件
2. **页面路由**：考虑实现简单的页面路由机制
3. **状态共享**：页面间状态共享和数据同步
4. **性能优化**：优化页面加载速度和内存使用

## 与其他模块的关系

- **组件模块**：使用通用组件构建页面界面
- **API 模块**：调用 API 获取数据和执行操作
- **工具模块**：使用工具函数处理页面逻辑
- **实体模块**：使用类型定义确保数据一致性
