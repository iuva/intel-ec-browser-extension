## 🏗️ 架构概览

### 模块关系图

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   background.ts  │◄───│   content.ts    │◄───│   components/   │
│   (后台脚本)     │    │   (内容脚本)    │    │   (Vue组件)     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         ▼                       ▼                       ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│     api/        │    │   entity/       │    │    utils/       │
│   (API接口)     │    │   (数据实体)    │    │   (工具函数)    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 数据流向

1. **用户操作** → 组件 → 内容脚本 → 后台脚本 → API 调用
2. **API 响应** → 后台脚本 → 内容脚本 → 组件 → 界面更新
3. **VNC 连接** → 组件 → 内容脚本 → 后台脚本 → 原生消息 → RealVNC
