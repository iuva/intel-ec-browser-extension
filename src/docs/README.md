# CPU 测试浏览器扩展 - 源代码目录结构说明

## 目录概述

`src` 目录包含了浏览器扩展的所有源代码文件，采用模块化设计，便于维护和扩展。

## 目录结构说明

### 📁 api/
**API 接口模块**
- 包含与后端服务通信的 API 接口定义
- 提供统一的 HTTP 请求封装

### 📁 components/
**Vue 组件模块**
- 包含所有可复用的 Vue 组件
- 采用组件化设计，提高代码复用性

### 📁 entity/
**数据实体模块**
- 定义 TypeScript 接口和数据类型
- 提供统一的数据模型定义

### 📁 img/
**图片资源模块**
- 存储扩展使用的图片资源
- 包括图标、加载动画等

### 📁 pages/
**页面模块**
- 包含扩展的各个页面组件
- 如弹出窗口页面等

### 📁 utils/
**工具函数模块**
- 提供通用的工具函数和工具类
- 包括 HTTP 请求封装、通知工具等

### 📄 根目录文件
- **background.ts** - 后台脚本，处理跨域请求和 VNC 连接
- **content.ts** - 内容脚本，在网页中注入 Vue 应用
- **manifest.json** - 扩展配置文件
- **popup.html** - 弹出窗口 HTML 模板
- **popup.ts** - 弹出窗口 TypeScript 逻辑
- **vite-env.d.ts** - TypeScript 环境类型定义

## 开发规范

1. **模块化设计**：每个目录都有明确的职责分工
2. **类型安全**：使用 TypeScript 确保类型安全
3. **响应式设计**：支持不同尺寸的浏览器窗口
4. **跨浏览器兼容**：兼容 Chrome、Firefox、Edge 等主流浏览器

## 扩展功能

- ✅ VNC 远程连接管理
- ✅ 主机列表展示和操作
- ✅ 用户认证和授权
- ✅ 实时通知系统
- ✅ 跨域请求处理
